How it works

A quick guide to preparing your file and reading the results.

1. Prepare your spreadsheet

Use one row per performance. The first row must contain these exact headers:

  • NAME
  • ID
  • COLLEGE
  • GENDER
  • EVENT NAME
  • PERFORMANCE TYPE
  • RESULT
ColumnAccepted values
GENDERMale / Female / M / W / Men / Women
EVENT NAMEFriendly (Long Jump, Shot Put) or code (LJ, 100m)
PERFORMANCE TYPETIME or DISTANCE
RESULT10.87, 2:05.31 (min:sec), 6.72 (metres)
TIMING (optional)FAT (default) or HAND

2. How scores are calculated

  • Each result is scored with the official World Athletics Scoring Tables (2025).
  • Track events: a faster time scores higher. Field events: a longer mark scores higher.
  • A performance between two table entries takes the lower score.
  • An athlete is identified by NAME + ID + COLLEGE; scores from all their events are summed into one total.
  • A college's score is the sum of its athletes' totals.

3. Invalid rows

Rows with an unknown event, an unreadable result, or a missing field are skipped and listed on the Rejected tab with a reason — the rest of your athletes are still scored.
