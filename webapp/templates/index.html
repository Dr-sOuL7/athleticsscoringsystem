Results file columns

The first row of your results file must contain these headers:

  • BIB NUMBER
  • GENDER
  • EVENT NAME
  • PERFORMANCE TYPE
  • RESULT

BIB NUMBER identifies the athlete. An optional TIMING column (FAT/HAND) is supported.

Optional roster columns

  • BIB NUMBER
  • NAME
  • ID
  • COLLEGE

Only BIB NUMBER is required; NAME/ID/COLLEGE may be partially missing. Each bib must appear at most once.

Result formats

10.87seconds
2:05.31min:sec
6.72metres

Events accept friendly names (Long Jump) or codes (LJ).
